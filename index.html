Google Sheets API
Lấy từ URL: docs.google.com/spreadsheets/d/ID/edit
API Key: vào Google Cloud → Credentials → API Key → giới hạn Sheets API.
Service Account JSON: dán nguyên nội dung file JSON (cho phép ghi IP lên Sheet).
Cache local
Hướng dẫn nhanh
1. Nhập Spreadsheet ID (cùng sheet với PC)
2. Nhập API Key (chỉ xem) hoặc Service Account JSON (xem + ghi IP)
3. Nhập tên thiết bị để phân biệt
4. Lưu → Về trang chủ → Kiểm tra IP

💡 Mở bằng Chrome → menu ⋮ → "Thêm vào màn hình chính" để dùng như app.
